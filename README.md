## Анализ коммерческой эффективности и операционной деятельности FMCG.

1. Описание проекта 
- Реализация цикла сквозной аналитики для крупного FMCG-производителя. Проект охватывает финансовый аудит маржинальности, оценку эффективности маркетинговых инвестиций (ROI), региональное развитие и стратегическое планирование продаж.
2. Ключевые аналитические блоки 
- Декомпозиция отклонений фактической выручки от плановой. Расчет чистого эффекта от изменения цены и влияния изменения физического объема продаж на общую прибыль.
- Анализ влияния инфляции производственных затрат (COGS) на валовую прибыль. Выявление драйверов роста доходности в условиях рыночной волатильности.
- Расчет прибыльности и окупаемости промо-акций. Формирование рекомендаций по оптимизации торговых инвестиций и пересмотру объемов отгрузок для достижения целевой рентабельности.
- Анализ выполнения таргетов и динамики роста (YoY) в разрезе регионов РФ. 
- Реализация модели пропорционального распределения утвержденного годового плана с использованием расчетных коэффициентов и сохранением исторических долей.
3. ETL и обработка данных 
- Power Query (M): Использование инструмента для автоматизации сбора данных, очистки справочников от дубликатов SAP-кодов и нормализации артикулов.
- Идентификация и аудит «потерянных» продаж (пустых категорий) из-за некорректного ведения мастер-данных.
- Построение сводных таблиц и использование функций сопоставления данных (XLOOKUP/VLOOKUP) для интеграции разрозненных источников.
4. Инструментарий
- Microsoft Excel
- Power Query
